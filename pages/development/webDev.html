Specifications and Documents

Hi Noobs!

Web development is an already complex system despite being one of the easiest "coding" to learn. However, all websites have basics structure and this includes html, css and javascript. Everything else you see on a website is a bonus.

With internet nowadays, we have so many info to choose from but sometimes we like a bit of structure so if you're overwhelmed and lost, I have a list of good sources for you to focus on.

*note that specification is not a document

you can think of document as a textbook and specification as a dictionary
